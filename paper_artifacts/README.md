## Paper artifacts (tracked)

This folder is intentionally small and stable.

It should contain only:
- A single selected PURE run summary (`comparison_summary.json`) plus its `metrics/*.json`
- 1–3 example documents showing: `source_requirements/`, `expanded_dialogues/`, `generated_requirements/`
- Pilot summary reports that are referenced in the paper

Everything else belongs in `data/outputs/` (ignored).
